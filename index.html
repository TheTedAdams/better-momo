Better MOMO

All you need to do is go to MOMO and click the bookmarklet. Easy as that!

Tired of clicking the bookmarklet?

Install Tampermonkey for Chrome, or Greasemonkey for Firefox and add the following UserScript:

// ==UserScript==
// @name        Better MOMO
// @namespace   http://tadams88.github.io/better-momo/
// @version     0.1
// @description Makes MOMO Better
// @match       http://momo/TimesheetMain.aspx*
// @match       http://momo/
// @copyright   2014+, Ted Adams
// ==/UserScript==

(function(a){
    var b = a.createElement('script');
    b.src = '//tadams88.github.io/better-momo/src/betterMomo.js';
    a.getElementsByTagName('head')[0].appendChild(b);
})(document);

Features

  • Time Tracking

    Click the play button next to any of your time inputs and, as long as you keep the window open, the script will automatically add your time spent.

    Switching to another task? Click the play button next to your new task and time will be added there. If a comment is required on your previous task the comment popup will appear.

    Finished for the day? Just hit the pause button next to your active task. If a comment is required the popup will appear, and then you can save.

  • Localized Current Day

    MOMO displays the current day as the current day in NZ (which is always a day off in the US). Better MOMO corrects this.

  • Updated Styles

    Modifies styling of the page so it isn't such a sprawling white expanse. (NOTE: Does not work in compatibility mode, which is turned on by default in Internet Explorer for intranet pages.)
